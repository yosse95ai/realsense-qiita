# realsense-qiita
RealSense D455を使用するためには，librealsense2というSDKが必要です．

そのために必要な環境構築や使用方法についてQiitaに記事を書いています．

https://qiita.com/RoaaaA

その際作成した記事のコードを保存しています．

# ブランチのルール

各ブランチについている数字がそのままシリーズ番号と一致しています．

> Ex：**03_06**：シリーズ(#03, #04, #05, #06)
> 
>     **07**：シリーズ(#07)
